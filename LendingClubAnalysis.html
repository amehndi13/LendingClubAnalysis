Evaluate Model with Business Context¶

Here we try to test the model in the actual context of the business. The following is the process I have taken to evaluate:

  1. Use the first half of our data for training and the second half for testing
  2. Fit a logistic regression on the training data
  3. Use this logistic regression to predict on the test set
  4. Take a subset of the loans that this model would approve
  5. Compare the number of loans given and the rate of return to what the business has already done

It seems like our model is slightly more strict in handing out loans. This however is a flawed statement on its own because we are only able to give our less loans in this situation and arent able to give out more loans.

We also note that the total return is ever so slightly less given our model. So we would need to further explore various techniques in order to improve on the current process.
